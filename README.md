# scramblr


## Install:

 - `git clone https://github.com/coder13/scramblr.git`
 - `npm install ./ -g`

OR

 - `npm install scramblr -g`

## Usage:

`scramble <type> [...args]`

## Scrambles:

 - 333
 - edges
 - LL
 - CMLL
 - LSLL
 - LSLLEO (LSLL with eo solved)
 - ZBLL
 - 2GLL
 - PLL
 - ZZLL
 - CLS
 - BLE
 - ZZ (takes arg as number of flipped edges)
 - LSE*
 - 2GEN*

 \* not random state
